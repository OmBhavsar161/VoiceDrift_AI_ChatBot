To Activate Virtual Environment:
.\rasa-env\Scripts\activate

To close Virtual Environment
deactivate 

1. rasa train

2. rasa data validate

3. rasa run --enable-api --cors "*"



For Debugging:
> rasa shell
> rasa shell nlu


Note: When ever making any changes in rasa just run all commands from '1.'



-----------For MVP Model-------------
Open 'server.py' and press run button to start backend server beacuse in MVP Flask is used as backend
